กำหนดค่า M และสตริง S งานของคุณคือหาว่า สับสตริงที่มีความยาวมากที่สุดซึ่งปรากฎใน S ไม่น้อยกว่า M ครั้ง มีความยาวเท่าไหร่

ข้อมูลนำเข้า

                บรรทัดแรกประกอบด้วยตัวเลข 2 ตัว N และ M ( 1 < N <15000, 2  < M < 20 )
                บรรทัดที่สองประกอบด้วยสตริง S ไม่มีช่องว่าง ความยาวเท่ากับ N

ข้อมูลส่งออก

                มีบรรทัดเดียวเป็นขนาดของสับสตริงที่ยาวที่สุดตามเงือนไขในโจทย์

 

หมายเหตุ

                ข้อมูลนำเข้า 10% มี N < 10
                ข้อมูลนำเข้า 30% มี N < 100
                ข้อมูลนำเข้า 50% มี N < 1000
                ข้อมูลนำเข้า 70% มี N < 10000

โจทย์โดย : กสิ ชนพิมาย (~Kasi~)
