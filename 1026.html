กำหนดให้ ลำดับเลขคณิตชุดหนึ่ง xi = (p * i) + q โดยที่ p และ q เป็นจำนวนเต็มค่าคงที่ และ x1, x2, …, xn เป็นจำนวนเต็มใดๆ

โจทย์
จงเขียนโปรแกรมเพื่อรับสายข้อมูลจำนวนเต็มชุดหนึ่ง แล้วหาว่า ส่วนของสายจำนวนเต็มที่รับเข้ามาซึ่งสอดคล้องตามเงื่อนไขที่กำหนดให้และมีความยาวของลำดับเลขคณิตยาวที่สุดนั้นเริ่มต้นที่ตำแหน่งใด และมีความยาวเท่าใด

ข้อมูลนำเข้า
บรรทัดแรกมีค่าจำนวนเต็ม n โดย (1 < n < 10000) ซึ่งเป็นจำนวนของสมาชิกของข้อมูลนำเข้าทั้งหมด และต่อจากนั้นรับค่า n บรรทัด โดยในแต่ละบรรทัดจะมีจำนวนเต็มจำนวนหนึ่งค่าซึ่งมีขอบเขตอยู่ในช่วง [-32768, 32767]

ข้อมูลส่งออก
บรรทัดแรก แสดงจำนวนเต็มสองค่า ค่าแรกเป็นตำแหน่งที่เริ่มของลำดับเลขคณิตที่ยาวที่สุดที่ปรากฎในสายข้อมูลนำเข้า และค่าที่สองเป็นความยาวของลำดับเลขคณิตที่พบในกรณีที่พบสายลำดับเลขคณิตที่ยาวที่สุดมากกว่าหนึ่งสาย ให้แสดงผลเฉพาะสายลำดับแรกที่พบ
